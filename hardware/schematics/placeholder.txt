================================================================================
UNIFIED 2-IN-1 LOW-COST ANALOG SIGNAL CONDITIONING SCHEMATIC
================================================================================
Target Production Budget: < $4.00 (Rs. 1,200 LKR)
Silicon Core: TL072 / TL082 Dual JFET-Input Operational Amplifier

--------------------------------------------------------------------------------
1. THE CORE PROCESSOR PINOUT (TL072 / 8-Pin DIP)
--------------------------------------------------------------------------------
                  ┌────▼────┐
   [pH Out] Pin 1 ┤1  ●    8├ VCC (+5V Main Power Input)
   [pH In-] Pin 2 ┤2        7├ [Turbidity Out] Pin 7 --> To MCU Analog Pin 35
   [pH In+] Pin 3 ┤3        6├ [Turbidity In-] Pin 6
   System GND     ┤4        5├ [Turbidity In+] Pin 5
                  └─────────┘

--------------------------------------------------------------------------------
2. STAGE A: ELECTROCHEMICAL pH AMPLIFIER (High-Impedance Buffer)
--------------------------------------------------------------------------------
* Shifts ±414mV scale to a completely positive, MCU-safe 1.5V to 3.5V window.

   +5V Rail ──────┬───────────────────────┐
                  │                       │
            [10k Resistor]          (Pin 8 - VCC)
                  │                       │
                  ├───────────────┐  ┌────┴────┐
 [STRIPPED CABLE] │               │  │  TL072  │
 ┌──────────────┐ │               └──┤3 (In+)  │
 │ Shield Mesh  ├─┼──────────────────┤         │
 ├──────────────┤ │       [10uF Cap] ├──┤2 (In-)  │
 │ Center Core  ├─┼───────┐          │  │         │
 └──────────────┘ │       │   ┌──────┤1 (Out)──┼──► To MCU Analog Pin 34
                  │       │   │      └─────────┘
            [10k Resistor]│   │
                  │       │   │
 System GND ──────┴───────┴───┴────────────────(Pin 4 - GND)


--------------------------------------------------------------------------------
3. STAGE B: OPTICAL TURBIDITY CONDITIONING (Differential Scattering Receiver)
--------------------------------------------------------------------------------
* Captures light blockage at 180° and light scattering at 90° using Channel 2.

                      +5V Rail ──────┬──────────────┐
                                     │              │
                               [10k Resistor] [10k Resistor]
                                     │              │
                                     ├──────────────┼──────────┐
                                     ▼              ▼          │
                                   [180° PT]      [90° PT]     │
                                     │              │          │
                                     ├──────────────┼──────┐   │
                                     │              │   ┌──┴───┴──┐
                                     │              │   │  TL072  │
                                     │              └───┤6 (In-)  │
                                     │                  │         │
                                     ───────────────────┤5 (In+)  │
                                                        │         │
                                                        │7 (Out)──┼──► To MCU 
                                                        └─────────┘   Pin 35
                                                             │
 System GND ─────────────────────────────────────────────────┴─────────
